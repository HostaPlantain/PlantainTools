音ハメVLOGメーカー.

BGMの音の強弱を解析し、各クリップの「音が合う瞬間」を尺・再生速度(0.8〜1.3倍)込みで自動マッチング。すべてブラウザ内で完結・自動保存。

🎬 クリップ 0本

動画ファイルをドロップ / クリックで選択（複数可）

切り出し位置や尺はアルゴリズムが決めるので、素材は長いままでOK。
同じクリップの別シーンが複数回使われることもあります。
🔊/🔇は書き出し動画にそのクリップの音を入れるかどうかです（解析には常に音を使います）。
「順番通り」モードではサムネイルのドラッグ順が使われます。

🎵 BGM

音楽ファイルをドロップ / クリックで選択
テンポ - BPM 尺 - 秒

BGM全体（末尾の無音は自動カット）をクリップで埋めます。
完成尺 = BGMの尺です。

🧩 音ハメ設計

音ハメ度 -

上段の色帯が各カット（数字=クリップ番号、×は再生速度）、下段の波形がBGMの音の強弱です。
カットをクリックすると別の候補（同スコア帯の別シーン/速度）に差し替えできます。

🎞️ プレビュー & 書き出し


⬇ ダウンロード

「動画に書き出す」は実時間再生しながら録画します（完成尺と同じ時間がかかります）。書き出し中はタブを閉じたり切り替えたりしないでください。
「⚡ ffmpegで高速書き出し」は解析結果を .bat ファイルに変換してダウンロードします。
素材の動画・BGMファイルと同じフォルダに置いてダブルクリックすると、ffmpeg が実時間を待たずに高画質（libx264 CRF17 / NVIDIA GPUがあれば自動でNVENC）で書き出します。
fmpeg 4.4以降のインストールが必要です（例: winget install Gyan.FFmpeg）。
