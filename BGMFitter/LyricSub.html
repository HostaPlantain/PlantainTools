🎬 MV歌詞字幕メーカー

動画と歌詞を読み込み → 自動同期 → 微調整 → .ass / .srt / burn.bat を書き出し
タップモード: 再生しながら Enter=次の行を開始　↓=行をここで終了　Backspace=1行戻る　Esc=終了
#開始終了歌詞
⏱=開始を現在位置に　▶=その行から再生　±0.1s で微調整。時刻は「1:23.45」形式で直接編集可。
フォント名(ASS) サイズ(1080p基準) 文字色 縁取り色 縁取り太さ 下端マージン 位置 行間ギャップ(秒)
使い方 / 焼き込み手順
  1. 動画と歌詞を読み込み、「行に分解して取り込む」
  2. 「自動同期」で音声の歌区間に歌詞を割り付け(目安。精度は音源次第)
  3. 「タップ打ち込み」で再生しながら Enter で正確に打ち直すのがおすすめ
  4. タイミング表で微調整(波形の縦線マーカーもドラッグ可)
  5. .ass と burn.bat を保存 → 動画と同じフォルダに置いて burn.bat を実行(ffmpeg が必要)
