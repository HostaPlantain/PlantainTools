1秒VLOGメーカー.

動画を集めて、BGMのリズムに合わせて約1秒ずつつなぐ。すべてブラウザ内で完結・自動保存。

🎬 クリップ 0本

動画ファイルをドロップ / クリックで選択（複数可）

サムネイルをドラッグで並べ替え。
各クリップのスライダーで「どの瞬間を使うか」を選べます。

🎵 BGM

音楽ファイルをドロップ / クリックで選択
テンポ - BPM

BGMのビートを自動検出し、カットの長さを曲のリズムにぴったり合わせます。
BGMなしの場合は1.00秒刻みになります。

使いたい曲を別で再生しながらリズムに合わせてタップするか、BPMを直接入力してください。
カット割りだけをそのテンポに同期し、書き出し動画のBGMは無音になります（音声ありのクリップの音は入ります）。
投稿時にYouTube等の公式機能でBGMを追加する使い方を想定しています。

1クリップ = - 秒

🎞️ プレビュー & 書き出し


⬇ ダウンロード

「動画に書き出す」は実時間再生しながら録画します（完成尺と同じ時間がかかります）。
書き出し中はタブを閉じたり切り替えたりしないでください。
「⚡ ffmpegで高速書き出し」は編集内容を .bat ファイルに変換してダウンロードします。
素材の動画・BGMファイルと同じフォルダに置いてダブルクリックすると、ffmpeg が実時間を待たずに高画質（libx264 CRF17 / NVIDIA GPUがあれば自動でNVENC）で書き出します。
ffmpeg 4.4以降のインストールが必要です（例: winget install Gyan.FFmpeg）。
